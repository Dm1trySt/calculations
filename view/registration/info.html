Для добавления пользователя: 1) Выберите из раскрывающегося списка пункт "Добавить пользователя". 2) Заполните поля login и password. 3) Нажмите "Выполнить". Для удаления пользователя: 1) Выберите из раскрывающегося списка пункт "Удалить пользователя". 2) В поле login или в поле id введите данные пользователя, которого хотите удалить. 3) Нажмите "Выполнить". Для изменения пользователя: 1) Выберите из раскрывающегося списка пункт "Изменить данные пользователя". 2) Заполните поле id и заполните поля с данными, которые хотите изменить. 3) Нажмите "Выполнить". ОБРАТИТЕ ВНИМАНИЕ! Все пароли в таблице представлены в виде хеш-строки md5 при входе, регистрации и изменении пароля, вводите его БЕЗ шифрования.
